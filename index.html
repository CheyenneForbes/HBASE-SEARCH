HBase-Search by jasonrutherglen

HBase with Lucene Search added.

With this system, the Lucene index files are explicitly stored and read from HDFS. Meaning, HDFS is not just used for archive or backup purposes, it's used as the main file system for Lucene. This provides automatic backup and recovery of the index in the event of [somewhat] common hardware failure.
